Propriedades CSS

Display Block

Serve para o elemento ocupar toda a largura disponível do seu contêiner e iniciar uma nova linha, empilhando-se verticalmente.

Display Inline

Faz com que os elementos permaneçam no fluxo do texto, ocupando apenas a largura necessária para o seu conteúdo, sem forçar uma quebra de linha. Elementos inline não respeitam definições de altura (height) e largura (width), e as margens superior e inferior não funcionam.

Display Inline-block

Semelhante ao inline, posiciona elementos lado a lado sem iniciar uma nova linha, mas permite modificações de largura (width) e altura (height).

Display Flex

Transforma um elemento em um contêiner flexível, permitindo o alinhamento automático de elementos filhos em uma linha horizontal por padrão.

Display Inline-Flex

Comporta-se como um elemento em linha (inline), mantendo as propriedades flexíveis (flex).
