Propriedades CSS

Display Block

Serve para o elemento ocupar toda a largura disponível do seu contêiner e iniciar uma nova linha, empilhando-se verticalmente.

Display Inline

Faz com que os elementos permaneçam no fluxo do texto, ocupando apenas a largura necessária para o seu conteúdo, sem forçar uma quebra de linha. Elementos inline não respeitam definições de altura (height) e largura (width), e as margens superior e inferior não funcionam.

Display Inline-block

Semelhante ao inline, posiciona elementos lado a lado sem iniciar uma nova linha, mas permite modificações de largura (width) e altura (height).

Display Flex

Transforma um elemento em um contêiner flexível, permitindo o alinhamento automático de elementos filhos em uma linha horizontal por padrão.

Display Inline-Flex

Comporta-se como um elemento em linha (inline), mantendo as propriedades flexíveis (flex).

Display Grid

Transforma um elemento em um contêiner de grade CSS, permitindo organizar elementos filhos em linhas e colunas com alto controle de layout.

Display Inline grid

Serve para criar um contêiner de grade para organizar elementos internos, mas o contêiner se comporta como um elemento em linha, ficando na mesma linha que outros elementos ao redor.

Display Flow-root

Faz o elemento gerar uma caixa de bloco que cria um novo contexto de formatação de bloco, passando a ser a raiz onde o layout dos elementos internos é organizado.

Display Table

Faz com que um elemento se comporte como uma tabela HTML.

Display Block flow

Faz com que um elemento se comporte como uma tabela HTML, passando a funcionar e organizar seu conteúdo como uma tabela.

Display Block flex

O display: block faz o elemento ocupar toda a largura disponível do seu contêiner, e o display: flex transforma um elemento em um contêiner flexível. Não é possível utiliza-los no mesmo elemento juntamente.

Position: Static

Valor padrão para qualquer elemento da página. Garantirá que ele seguirá o fluxo de renderização padrão atribuído pela propriedade display.

Position: Relative

Ao aplicarmos o tipo relative, a posição deste elemento poderá ser manipulada através das propriedades Top, Bottom, Left, Right. Estes por sua vez assume valores numéricos que determinaram em qual coordenada X ou Y seu elemento irá se mover.

Position: Absolute

O position absolute garante que você consiga posicionar qualquer elemento de sua página de acordo com o elemento pai que tenha um posicionamento diferente de static.

Positon: Fixed

Este tipo de posição se comporta muito similar ao position absolute, com a única diferença que sua referência de posição é ao elemento window/body e não ao elemento pai no qual este está inserido.

Position: Sticky

Permite criarmos elementos que dependendo de sua posição setada, hora funcionam como relative e outras como fixed. É um recurso muito interessante para utilização de menus que são fixos e ao mesmo tempo relativos quando se é necessário.
