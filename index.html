Display Block flow

O block define o comportamento externo do elemento, fazendo com que ele ocupe toda a largura disponível e comece em uma nova linha, enquanto o flow define o comportamento interno, indicando que os elementos filhos serão organizados automaticamente seguindo o fluxo normal do HTML. Na prática, a sintaxe display: block flow é equivalente ao display: block, sendo apenas uma forma mais moderna de escrever a mesma regra.

Display Block flex

Indica que o elemento é exibido como um bloco e utiliza o layout flexível. O block faz com que o elemento ocupe toda a largura disponível e comece em uma nova linha, enquanto o flex permite organizar e alinhar os elementos filhos de forma flexível dentro do container.

Display Block grid

O elemento é exibido como um bloco e utiliza o layout em grade. O block faz com que o elemento ocupe toda a largura disponível e comece em uma nova linha, enquanto o grid permite organizar os elementos filhos em linhas e colunas dentro do container. Essa sintaxe é equivalente ao uso de display: grid no CSS.

Display Block flow-root

Faz com que o elemento seja exibido como um bloco e criará um novo contexto de fluxo para o layout. O block faz com que o elemento ocupe toda a largura disponível e comece em uma nova linha. Já o flow-root faz com que o elemento organize seu próprio fluxo interno de conteúdo, evitando interferências de elementos flutuantes externos. Essa combinação está relacionada ao comportamento de display: flow-root no CSS.

Display Inline flow

O elemento é mostrado no formato inline, permanecendo na mesma linha que outros elementos. O termo flow indica que o conteúdo interno segue o fluxo normal do documento, sendo organizado automaticamente conforme a ordem em que aparece no HTML.

Display Inline flex

Faz o elemento se comportar como inline, definindo o comportamento externo do elemento, permitindo que ele fique lado a lado com outros sem quebrar a linha. Já o valor flex define o comportamento interno, organizando e alinhando os elementos filhos de forma flexível dentro do container.

Display Inline grid

O elemento se comporta como inline, permanecendo na mesma linha que outros elementos, e o valor grid permite organizar os elementos filhos em linhas e colunas dentro do container. Faz com que o elemento fique alinhado com outros na mesma linha, enquanto seu conteúdo interno é organizado em forma de grade.

Display Inline flow-root

O valor inline define o comportamento externo do elemento, permitindo que ele fique lado a lado com outros sem quebrar a linha. Já o valor flow-root define o comportamento interno, criando um novo contexto de formatação que contém elementos flutuantes (floats) e permite definir largura e altura. Essa sintaxe funciona da mesma forma que o display: inline-block.
